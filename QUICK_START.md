## 🎨 如何调整圆弧范围

编辑 `src/main.rs`，修改这两行：
```rust
let arc_start = 0.0;    // 起始角度
let arc_end = 180.0;    // 结束角度
```

**常用角度范围**：
- `0° - 180°`：上半圆（当前设置）
- `180° - 360°`：下半圆（只会看到另一个交点）
- `0° - 360°`：完整圆（会看到2个交点）
- `0° - 90°`：右上四分之一圆（只有1个交点）
- `90° - 180°`：左上四分之一圆（只有1个交点）

## 💡 提示

- 两个完整圆相交时有2个交点
- 将其中一个改为圆弧后，只有在圆弧范围内的交点会被显示
- 当前设置（0° - 180°）只显示上方的交点
- 如果改为 180° - 360°，则只会显示下方的交点
